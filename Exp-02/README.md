### ğŸ¨ **1. Inline CSS**  
- Applied directly to an HTML element using the `style` attribute.

  ```html

  ```

### ğŸ¨ **2. Internal CSS**  
- Defined inside a `<style>` tag within the `<head>` section.  
- Example:  
  ```html

          background-color: lightcoral;
          color: white;
      }
  </style>
  ```

### ğŸ¨ **3. External CSS**  
- Stored in a separate `.css` file and linked in the `<head>`.

  ```css
      font-size: 18px;
  }
  ```
